Archivos recomendados:

- `miguel-photo.jpg` — Foto de perfil (retrato), recomendado 800x800 px o cuadrada; se mostrará en la cabecera y hero.

- `logo-sena.png` — Logo del SENA (PNG transparente), tamaño aproximado 300x100 px.

Imágenes para cursos (colócalas también en `assets/`):

- `course-matematicas.jpg` — Imagen representativa para "Matemáticas básicas" (recomendado 1200x800 o 800x600).
- `course-calculo.jpg` — Imagen representativa para "Cálculo diferencial" (recomendado 1200x800 o 800x600).
- `course-programacion.jpg` — Imagen representativa para "Fundamentos de programación" (recomendado 1200x800 o 800x600).

Los nombres deben coincidir exactamente con los indicados para que las tarjetas muestren las imágenes. Si quieres, puedo traer imágenes libres de derechos y guardarlas aquí como placeholders; dime si prefieres eso.

Imágenes para la sección "Quién soy" (colócalas en `assets/`):

- `studies.jpg` — Ilustración o foto alusiva a la educación/formación.
- `idiomas.jpg` — Icono o ilustración para idiomas.
- `intereses.jpg` — Imagen que represente intereses personales (música, lectura, programación).
- `herramientas.jpg` — Imagen o collage que represente herramientas técnicas (Python, Excel, R, GitHub).

Coloca estos archivos en `assets/` con exactamente esos nombres para que se muestren en la página "Quién soy".

Notas:
- Los nombres de archivo deben coincidir exactamente con los anteriores para que las páginas los referencien automáticamente.
- Formatos: SVG (logo vectorial) o PNG (transparente) para logos, JPG o PNG para fotografía.
- Si quieres que traiga imágenes de ejemplo (placeholder públicas) indícame si prefieres fotos realistas o iconos institucionales; puedo incluir URLs de ejemplo para que las descargues.
